# ML-Regression-Model---Melbourne-Housing-----Dataset
The melb_data.csv file contains detailed records of residential property sales in Melbourne, Australia. It includes 13,580 entries and 21 columns with features

Property type (house, unit, etc.)

Number of rooms, bathrooms, and car spaces

Land and building size

Sale price and date

Geographic location (latitude, longitude, suburb, region)

Council and postcode information

Distance from the city center

This dataset is ideal for real estate price prediction, urban development analysis, and geospatial modeling.
